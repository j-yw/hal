# Ralph Progress Log

## Codebase Patterns

- Factory `ArtifactReference` keeps `name` and `type` as required stable fields while richer artifact metadata is additive and optional; keep legacy `path` for current CLI compatibility and use `sourcePath`/`storedPath` for durable collection metadata.
- Factory artifact payloads belong under the global factory store at `artifacts/<run-id>/`; use `Store.SaveArtifactFile` and `Store.ResolveArtifactPath` so writes stay scoped to the store and never create project `.hal` artifact state.
- Local factory artifact collection keeps legacy display `path` values for CLI compatibility while copying payloads into the store with stable `sourcePath` and `storedPath` metadata.
- Factory status/doctor snapshots should use structured package results (`status.Get`, `doctor.Run`) behind injectable deps, then flow through normal artifact persistence instead of parsing CLI output.
- Factory PR/CI outcome artifacts are materialized from `compound.CIState` in `.hal/auto-state.json`; persist safe PR metadata during `runPRStep` and record partial warning artifacts when outcome state is unavailable.
- Sandbox artifact collection uses `factory.SandboxArtifactCopier` plus `factory.CollectSandboxArtifacts`; remote paths are copier inputs only, while persisted metadata keeps safe display paths and store-backed paths.
- Sandbox factory runs collect remote artifacts during `recordFactoryRunArtifacts` when the persisted run record has `ExecutorModeSandbox`; tests should mutate the run record inside the fake pipeline and inject `sandboxCopier`/`sandboxRequests` through `factoryRunDeps`.
- Read-only factory subcommands should use small injectable deps structs with `defaultStore func() (factory.Store, error)` so command tests can use isolated `factory.NewStore(t.TempDir())` roots.
- Factory artifact JSON read surfaces should use dedicated safe response structs, not raw `factory.ArtifactReference`; omit `sourcePath`/`url`, keep display `path` plus store-relative `storedPath`, and sanitize summary/warning strings that contain secrets or raw IP addresses.
