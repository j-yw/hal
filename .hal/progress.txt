# Ralph Progress Log

## Codebase Patterns

- Factory run-scoped secrets should model requested env-backed secrets with `internal/factory.RunSecretInput` and persist only `RunSecretMetadata` on `RunRecord.Secrets`; durable records should contain `name`, `source`, `required`, and `present`, never raw secret values.
- Resolve factory run secrets through `factory.ResolveRunSecrets` during `executeFactoryRun` setup before marking runs in progress or calling local/sandbox executors; keep raw values only on in-memory `ResolvedRunSecret` values.
- Future factory run secret sources should plug into `internal/factory.RunSecretProvider` via `ResolveRunSecretsWithProviders`; keep the default `ResolveRunSecrets` path env-only until a source is implemented.
- Factory type-contract changes should update `internal/factory/types_test.go` JSON tag, round-trip, explicit field, optional omission, and legacy JSON loading coverage.
