## Codebase Patterns
- `runInit` accepts `cmd *cobra.Command` which may be nil in tests ‚Äî guard flag reads with `if cmd != nil`
- Use `template.DefaultFiles()` to iterate the 3 core templates (prompt.md, progress.txt, config.yaml)
- Extracted functions follow `run<Command>` naming and accept `io.Writer` for testability
- GPG signing may not be available ‚Äî use `git -c commit.gpgsign=false` when needed
- Backup files use timestamped format `<file>.YYYYMMDD-HHMMSS.bak` via `time.Now().Format("20060102-150405")`
- `refreshTemplates(halDir, dryRun, w)` is the testable function for template refresh ‚Äî called from `runInit` via `doRefresh` flag
- Test helpers (`writeFile`, `writePRD`) are shared within the `cmd` package via `archive_test.go` ‚Äî don't redeclare them
- For Cobra-level tests where the handler writes to `os.Stdout` directly, use `os.Pipe()` redirect to capture output (Cobra's `SetOut` won't work)
