## hal prd

Manage PRD files

### Synopsis

Inspect and manage Product Requirements Document files.

Use 'hal prd audit' to check PRD health and detect drift.

```
hal prd [flags]
```

### Examples

```
  hal prd audit
  hal prd audit --json
```
