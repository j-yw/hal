## hal sandbox

Manage Daytona sandboxes

### Synopsis

Manage Daytona sandbox environments for isolated development.

Subcommands:
  setup       Configure Daytona API credentials
  start       Create and start a sandbox
  stop        Stop a running sandbox
  status      Show sandbox status
  delete      Delete a sandbox
  shell       Open an interactive shell
  exec        Run a command in the sandbox
  snapshot    Manage sandbox snapshots

### Examples

```
  hal sandbox setup
  hal sandbox start
  hal sandbox status
```

### Options

```
  -h, --help   help for sandbox
```
