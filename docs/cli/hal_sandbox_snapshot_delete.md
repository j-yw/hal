## hal sandbox snapshot delete

Delete a snapshot

### Synopsis

Delete a Daytona sandbox snapshot.

Requires --id flag specifying the snapshot ID to delete.

```
hal sandbox snapshot delete [flags]
```

### Examples

```
  hal sandbox snapshot delete --id snap-123
```

### Options

```
  -h, --help        help for delete
      --id string   snapshot ID to delete (required)
```
